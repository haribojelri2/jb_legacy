# JB Legacy — 시연 영상 대본 (v2)

**목표 러닝타임:** 약 3분 30초 (대회 규정 최대 5분 이내)
**형식:** 실제 앱 조작 라이브 녹화(마우스 커서 표시) + OpenAI TTS 한국어 나레이션.
**싱크 원칙:** 씬을 잘게 쪼개(14개) 각 씬의 화면 동작이 나레이션 내용과 같은 순간에 보이도록 씬 단위로 길이를 맞춘다. 긴 연산(분석·D안 생성·GAN)은 해당 씬 안에서 타임랩스(배속) 처리.
**시연 페르소나:** 이사장(62세·전주·30년 한정식, 월 순이익 450만, 승계 유리)

> 표기: **[화면]** = 실제 녹화 동작 / **[나레이션]** = TTS 대사

---

## 씬 1. 오프닝 — 문제 제기 (~22s)
- **[화면]** 인트로 타이틀 카드 (다크 네이비)
- **[나레이션]** "전북과 광주에는 삼십 년 넘게 가게를 지켜온 사장님이 많습니다. 그런데 막상 은퇴할 때가 되면, 팔아야 할지, 자녀에게 물려줘야 할지, 세금은 얼마인지, 노후 자금은 버틸지, 아무도 한 번에 알려주지 않습니다. JB 레거시는 이 결정을 AI 멀티 에이전트가 통합 설계해 드립니다."

## 씬 2. 온보딩 — 페르소나·조건 입력 (~13s)
- **[화면]** 페르소나 목록에서 이사장 선택 → 세부 조건 expander 열어 확인 → 분석 시작하기 클릭
- **[나레이션]** "예순두 살 이사장님. 전주에서 삼십 년간 한정식집을 운영해 오셨습니다. 페르소나를 고르고, 월 순이익과 세부 조건을 확인한 뒤, 분석을 시작합니다."

## 씬 3. 질문 → 멀티에이전트 분석 (타임랩스) (~15s)
- **[화면]** 추천 질문 칩("폐업할지 승계할지 종합 분석") 클릭 → 분석 진행 상태 라벨(타임랩스)
- **[나레이션]** "무엇이 궁금한지 누르기만 하면 됩니다. 관리자 에이전트가 질문을 분석해 필요한 전문 에이전트를 동적으로 투입하고, 가치평가, 세금, 자산운용이 병렬로 실행됩니다."

## 씬 4. 에이전트 현황 + 추천 결과 (~11s)
- **[화면]** 에이전트 현황 펼침(투입 에이전트 목록) → 채팅의 추천 시나리오 응답
- **[나레이션]** "분석이 끝나면 에이전트 현황에서 투입된 에이전트를 확인할 수 있습니다. 추론은 클로드가, 계산은 결정론적 모듈이 담당합니다."

## 씬 5. 세금 비교 — 매각 vs 승계 (~18s)
- **[화면]** 세금 비교 탭 (매각 2,875만원 vs 특례 0원) 커서로 카드 순회
- **[나레이션]** "가게를 외부에 팔면 권리금 기타소득세가 약 이천팔백칠십오만 원. 하지만 자녀에게 물려주는 가업승계 과세특례를 적용하면 증여세는 영 원입니다. 시스템은 가업 십 년 요건까지 판정해서, 요건이 안 되면 일반 증여세로 자동 전환합니다."

## 씬 6. 노후 설계 — A/B/C + 생존확률 (~16s)
- **[화면]** 노후 설계 탭 클릭 → A/B/C 카드 → 스크롤로 JB 상품 배분·생존확률 차트
- **[나레이션]** "전북은행과 광주은행의 실제 상품으로 노후 포트폴리오를 구성하고, 몬테카를로 시뮬레이션으로 백 세까지 자산이 버틸 확률을 계산합니다. 목표 생활비에 못 미치면, 지속 가능한 금액을 자동으로 역산해 제시합니다."

## 씬 7. 가족 공유 — 문자 링크 전송 (~13s)
- **[화면]** 가족 협상 탭 → "이과장에게 공유 (문자 링크 전송)" 클릭 → 전송 확인 토스트 → "링크 열기" → 자녀 전체화면 리포트 스크롤
- **[나레이션]** "부모의 분석 결과는 자녀에게 문자 링크로 전송됩니다. 자녀는 자기 휴대폰에서 링크를 열어, 아버지의 은퇴 플랜 리포트를 확인합니다."

## 씬 8. 가족 협상 — D안 생성·추천 (~18s)
- **[화면]** 자녀 화면의 협상 폼: 승계 70%·자문료 20% 슬라이더 확인 + 한마디 입력 → "협상안 제안하기" → D안 생성(타임랩스) → 아버지 화면 복귀 → D안 섹션
- **[나레이션]** "자녀가 승계 비율 칠십 퍼센트와 자문료 이십 퍼센트를 제안하면, 부모의 노후와 가족 자산을 모두 반영한 가족 합의안 디가 만들어집니다. 이제 AI는 에이, 비, 씨, 디 네 가지 중 가장 균형 잡힌 하나를 추천합니다."

## 씬 9. 경영 건강 + 권리금 동적평가 (~15s)
- **[화면]** 경영 건강 탭(조기경보 대시보드) 스크롤 → 권리금 평가 탭(동적 재평가)
- **[나레이션]** "분석만이 아니라 일상도 지킵니다. 경영 건강 조기경보가 매출 흐름과 상권을 상시 점검해 폐업 위험을 미리 알리고, 권리금은 상권 데이터를 반영해 동적으로 재평가됩니다."

## 씬 10. 거래 안심 — 룰+LLM 하이브리드 (~14s)
- **[화면]** 거래 안심 탭: 소비흐름 → 이상거래 경보 → 🤖 AI 위험 판단 → "이과장에게 이상거래 알림 보내기" 클릭 → 전송 확인
- **[나레이션]** "은퇴 자산을 노리는 금융사기도 막습니다. 룰 기반으로 이상거래를 탐지하고, 언어모델이 보이스피싱과 명의도용 여부를 종합 판단해서, 즉시 취할 행동과 가족 알림까지 제안합니다."

## 씬 11. 청년 매칭 (~12s)
- **[화면]** 청년 매칭 탭: 인수 후보 카드(자기자금+JB 인수대출) → 대출 상품·인수 절차
- **[나레이션]** "매각을 결정한 경우에는, 폐업 대신 지역 청년 창업가에게 넘겨서 상권과 고용을 잇습니다. 부족한 인수 자금은 JB 인수 대출로 연결됩니다."

## 씬 12. AI 이중 검증 (GAN 실행) (~12s)
- **[화면]** AI 검증 탭 → "GAN 테스트 시작" 클릭 → 토론·채점(타임랩스) → 점수 결과
- **[나레이션]** "그리고 모든 답변은 금융소비자보호법 검수와, 공격자와 방어자가 토론하는 적대적 AI 검증을 통과해야 사용자에게 전달됩니다."

## 씬 13. 상담·리포트 — PDF·예약·음성 브리핑 (~14s)
- **[화면]** 상담·리포트 탭: 음성 브리핑 → 상담 예약 → PDF 리포트 다운로드 버튼
- **[나레이션]** "마지막으로, 분석 결과는 PDF 리포트로 출력해 PB 상담에 바로 활용하고, 세무사와 PB 상담 예약, 시니어를 위한 음성 브리핑까지 한 화면에서 이어집니다."

## 씬 14. 클로징 (~15s)
- **[화면]** 아웃트로 카드
- **[나레이션]** "세금부터 노후, 가족 합의, 사기 예방까지. JB 레거시는 사장님의 명예로운 엑시트를 설계하고, JB금융그룹의 수신과 여신 고객을 함께 만듭니다. 평생 일군 가게, 가장 명예롭게. JB 레거시."

## 제작 파이프라인 (v2)
1. TTS 생성(씬 14개) → 씬별 길이 측정
2. Playwright **연속 라이브 녹화**(1600×900, 가짜 커서 오버레이 주입) — 씬 경계 타임스탬프 기록
3. ffmpeg: 씬별 컷 → 나레이션 길이에 맞춰 리타이밍(긴 연산은 자연 타임랩스) → 나레이션 합성 → 페이드 연결
4. 프레임 추출 → 씬별 나레이션-화면 일치 검증 → 불일치 씬 재작업
