## 1. Summary — JB Legacy

**핵심 메시지:** JB Legacy는 은퇴를 앞둔 지역 자영업자의 매각·승계·노후자금·가족합의·사기위험을 한 번에 설계하는 AI 멀티에이전트 서비스다.

- 서비스명: JB Legacy — 자영업자 AI 은퇴·승계·엑시트 설계 어시스턴트
- 대상: 전북·광주 지역 50~65세 자영업자와 승계 당사자인 자녀
- 한 줄 가치: "평생 일군 가게의 명예로운 마무리와 JB의 수신·여신 기회를 동시에 만든다"
- 구현 상태: Streamlit MVP 구현 완료, 단위 테스트 51건, UI 자동화 테스트 9건

**시각요소:** 중앙 대형 로고 + 5대 가치 배지(세금, 노후, 가족, 거래안심, 상담연계)

---

## 2. Problem Definition — 은퇴 앞 자영업자의 4대 난제

**핵심 메시지:** 자영업자의 은퇴 의사결정은 세무·자산운용·가족승계·사기위험이 한꺼번에 얽혀 있어 기존 상담 방식으로는 빠르게 답하기 어렵다.

- 팔까, 물려줄까, 절충할까를 정량 비교하기 어렵다.
- 권리금·세금·노후 현금흐름을 한 화면에서 비교할 도구가 부족하다.
- 부모와 자녀가 승계 조건을 조율하는 과정에서 갈등이 발생한다.
- 고령 고객은 보이스피싱·명의도용 등 은퇴자산 대상 금융사기에 취약하다.
- PB 상담은 가치평가·세무·운용을 모두 다뤄야 해 시간과 인력 부담이 크다.

**시각요소:** 4분할 페인포인트 카드 + "상담 단절" 흐름도(세무사, PB, 가족, 은행 앱이 분리됨)

---

## 3. Solution Overview — AI 멀티에이전트 통합 설계

**핵심 메시지:** 질문 하나를 입력하면 LangGraph 멀티에이전트가 세금, 은퇴 포트폴리오, 가족 합의안, 거래 안심, 상담 예약까지 자동으로 연결한다.

- 4개 시나리오: A 완전매각, B 완전승계, C 절충안, D 가족합의안
- 판단: Supervisor가 질문 의도와 필요한 에이전트를 동적 선택
- 행동: RAG 조회, 계산, 가족 공유, 이상거래 알림, PB 상담 예약
- 검증: ComplianceGuard와 GANReview로 응답 품질과 규제 리스크 점검
- 결과: 고객에게는 쉬운 설명, PB에게는 상담 가능한 근거 자료 제공

**시각요소:** 사용자 질문 → AI Agent → A/B/C/D 추천 → 가족공유·PDF·상담예약으로 이어지는 한 장 플로우

## 4. Key Features — 주요 기능 정의

**핵심 메시지:** JB Legacy의 핵심 기능은 "계산은 결정론, 설명은 AI, 검증은 이중화" 원칙으로 설계되어 있다.

| 기능 | 본선 구현 내용 |
|---|---|
| 사업체 가치평가 | 월 순이익, 업력, 보증금, 시설비 기반 권리금·총 가치 산출 |
| 세금 시뮬레이션 | 외부 매각, 가업승계 특례, 일반 증여 세금 비교 |
| 노후 설계 | JB 실제 상품 기반 월 수령액, 100세 생존확률, 지속가능 생활비 역산 |
| 가족 협상 D안 | 자녀 제안 조건을 반영해 A/B/C와 동일 기준으로 비교 |
| 거래 안심 | 고액·심야·분할·해외 거래 감지와 가족 알림 |
| 시니어 UI | 큰 글씨, 쉬운 표현, 음성 입력·브리핑 |
| 검증 루프 | 금소법 검수와 적대 검증으로 응답 개선 |

**시각요소:** 기능별 아이콘 그리드 + MVP 화면 캡처 2개(세금/노후)

## 5. User Scenario — 이사장 은퇴·승계 시나리오

**핵심 메시지:** 대표 페르소나 이사장은 "딸에게 넘길지, 팔지"라는 자연어 질문만으로 은퇴·승계 의사결정 자료를 얻는다.

- 조건: 62세, 전주 한정식 30년 운영, 은퇴 3년 이내, 목표 생활비 월 300만원
- 질문: "가게를 딸한테 넘길지 팔지 모르겠어요"
- 분석: 사업체 가치 3억 1,200만원, 매각세 약 2,875만원, 가업승계 특례 0원
- 노후: A/B/C/D 월 수령액과 100세 생존확률 비교
- 가족: 자녀가 70% 승계와 자문료 20%를 제안하면 D안 자동 생성
- 마무리: 가족 공유, PDF 리포트, 전북은행 WM 상담 예약

**시각요소:** 6단계 고객 여정 타임라인 + 부모 화면/자녀 화면 2컷

## 6. Data & Tech Plan — 데이터 및 기술 활용

**핵심 메시지:** 본선 MVP는 공식 세율·실존 JB 상품·결정론 계산기를 중심으로 동작하며, 보조 기능은 데모 데이터임을 명확히 고지한다.

| 구분 | 활용 |
|---|---|
| 오케스트레이션 | LangGraph 기반 Supervisor, Profiler, 병렬 fan-out, Synthesizer |
| LLM | Claude 계열 분석·라우팅·검수, 팩토리 구조로 GPT 계열 전환 가능 |
| RAG | ChromaDB + OpenAI Embedding으로 세법 근거 검색 |
| 계산 엔진 | 세금, 포트폴리오, 몬테카를로 생존확률은 Python 결정론 모듈 |
| 데이터 | 공식 세율, 실존 JB 상품, SQLite 데모 거래·매출·청년 후보 데이터 |
| UI/산출물 | Streamlit, PDF 리포트, 가족 공유, 상담 예약 플로우 |

- 향후 연동: JB카드 마이데이터, 전북은행 거래내역 조회, JB Legacy 청년 후보 DB, 자동이체 API
- 고지: 거래 안심은 실시간 FDS 대체가 아니라 고객 동의 기반 보조 감시·가족 통지 기능

**시각요소:** 시스템 아키텍처 다이어그램 + "실데이터 계산 / 데모 데이터 / 향후 연동" 구분 라벨

## 7. 본선 구현 결과 — 실제 동작하는 MVP

**핵심 메시지:** 아이디어 수준이 아니라, 입력부터 분석·검수·공유·예약까지 본선에서 직접 시연 가능한 MVP로 구현했다.

- 구현 완료: 채팅 분석, 세금 카드, 노후 포트폴리오, 생존확률, 가족 D안, 거래 안심, 청년 매칭, PDF 리포트, 상담 예약
- Agent 구현: Supervisor, Profiler, BusinessValuation, TaxSuccession, PostExitWM, Negotiation, Synthesizer, ComplianceGuard, GANReview
- 품질 검증: 단위 테스트 51건, UI 자동화 테스트 9건 통과
- UI 개선: 발표자료 배경과 동일한 다크 네이비 톤으로 앱 화면 정비
- 내부통제: 금소법 5규칙 검수, 수치 계산-응답 정합성 검증

**시각요소:** 실제 Streamlit 화면 캡처 3장 + 테스트 결과 배지

## 8. Expected Impact — 기대 효과와 JB 사업 연계

**핵심 메시지:** 고객에게는 은퇴 의사결정 시간을 줄이고, JB에는 은퇴자산 수신과 청년 인수자 여신을 동시에 만든다.

| 관점 | 기대 효과 |
|---|---|
| 고객 | 복잡한 세금·노후자금·가족협상을 수 분 내 비교 |
| PB | 상담 전 자료 자동 생성으로 준비 시간과 반복 설명 감소 |
| 리스크 | 금소법 검수와 이상거래 알림으로 불완전판매·사기위험 완화 |
| JB | 부모 매각대금 수신, 청년 인수대출 여신, WM 상담 전환 기회 창출 |
| 지역 | 폐업 대신 청년 승계로 상권·고용 유지 |

**시각요소:** 부모 자산 → JB 상품 → 청년 인수대출 → 지역 상권으로 이어지는 순환 다이어그램

## 9. 확장성 — PoC에서 JB 모바일 서비스까지

**핵심 메시지:** 현재 MVP는 PB 보조 도구로 바로 파일럿이 가능하며, 향후 JB 모바일 앱의 라이프케어 AI 메뉴로 확장할 수 있다.

- 1단계 PoC: 현재 Streamlit 본선 데모
- 2단계 파일럿: 전북은행 PB 창구 5개 지점 태블릿 상담 보조
- 3단계 내부 적용: PB 상담 리포트 자동화, 세무 제휴 네트워크 연계
- 4단계 고객 서비스화: JB 모바일 앱 메뉴 통합, 마이데이터 연동
- 확장 고객군: 농업인 은퇴, 소규모 법인 대표, 광주·전남 자영업자, 30~40대 장기 은퇴 설계

**시각요소:** 확장 로드맵 4단계 + 계열사 연결 맵(전북은행, 광주은행, JB우리캐피탈, JB자산운용)

## 10. 본선 시연 계획 & Closing

**핵심 메시지:** 본선 시연은 "질문 1개 → 에이전트 분석 → 세금·노후·가족·거래안심 → 상담 예약"의 완성 흐름을 3분 안에 보여준다.

**시연 순서**

1. 이사장 페르소나로 자연어 질문 입력
2. 에이전트 상태 패널에서 LangGraph 분석 흐름 확인
3. 세금 비교와 노후 월수령·생존확률 확인
4. 딸의 협상 조건을 반영해 D안 생성
5. 거래 안심 알림과 가족 공유 확인
6. PDF 리포트와 PB 상담 예약으로 마무리

**클로징 문장:** "JB Legacy는 사장님의 명예로운 엑시트를 설계하고, JB의 새로운 수신·여신 고객을 함께 만듭니다."

**시각요소:** 데모 흐름 6컷 스트립 + 마지막 JB Legacy 로고
